## Best Xcode Shortcuts

- ⌘ + L - jump to line
- ⌥ + ⌘ + E: edit scoop in place
- ⇧ + ⌃ + ⌘ + A - Show git story
- ⌥ + ⌘ + [ - move line up
- ⌥ + ⌘ + ] - move line down
